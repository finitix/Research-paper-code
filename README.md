

🔹 What is this project?

This project is an AI-based decision-support system that helps detect risky or misleading design patterns in online interfaces like websites and apps.

Sometimes, websites use tricks like countdown timers, pre-selected options, or hidden information to influence user decisions. These are called dark patterns, and they can make users take actions they didn’t fully intend.

Our system helps identify these risky patterns and warns users or auditors before making important decisions.

🔹 Why is this project important?

In today’s digital world, people make many decisions online such as buying products, subscribing to services, or accepting terms and conditions.

Many users do not notice hidden tricks in interfaces, which can lead to:

Wrong decisions
Financial loss
User regret

This project helps solve that problem by making interfaces more transparent and safe.

🔹 How does the system work?

Our system works in a step-by-step process:

Input Data
Takes data from website interface
Includes text, design elements, and user behavior
Feature Extraction
Extracts useful information like clicks, time, layout, etc.
Risk Detection (Hybrid Model)
Rule-based system detects clear patterns
Machine learning detects complex patterns
Decision Output
Shows result as:
High Risk ⚠️
Safe ✅
Uncertain ❓
🔹 What makes this project unique?
Uses hybrid model (Rule + ML)
Provides explainable results (not black-box)
Includes confidence system (avoids wrong predictions)
Works as a decision-support tool, not fully automatic
🔹 What data is used?

We used a synthetic dataset (artificial data) that simulates real user behavior.

It includes:

Click patterns
Time spent on page
User actions
Interface signals

This helps train the model without privacy issues.

🔹 What are the results?

The model performs very well:

High accuracy
Very high precision
Almost zero false positives

This means the system is reliable and trustworthy.

🔹 Where can this be used?

This system can be used in:

E-commerce websites
Subscription platforms
Banking and finance apps
Government and legal websites

It helps ensure fair and safe user experience.

🔹 Final Conclusion

This project shows that AI can be used not just for automation, but also for protecting users and improving decision-making.

Instead of replacing humans, our system supports humans by highlighting risky situations and allowing better, informed decisions.
